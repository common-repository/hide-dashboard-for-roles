Allows administrators to block access to Wordpress' admin dashboard based on the user's role.

== Description ==

Prevents access to the Admin Dashboard, redirecting users to the site's homepage.
Send all support requests to hello@sagrev.solutions.

== Installation ==

1. Upload the contents of the zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
